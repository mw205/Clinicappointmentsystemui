
# Clinic Appointment System UI

This is a Vue 3 + Vite code bundle for Clinic Appointment System UI. The original project is available at https://www.figma.com/design/6SZZdAgXXTjtqoWVosX50D/Clinic-Appointment-System-UI.

## Running the code

Run `pnpm install` to install the dependencies.

Run `pnpm dev` to start the development server.

Run `pnpm typecheck` to validate the Vue TypeScript setup.
